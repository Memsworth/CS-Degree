# Object-Oriented Software Design - Course Syllabus

## **Unit 1: Basics of Object-Oriented Programming and Design**
- Introduction to Object-Oriented Software Design  
- Comparison of Programming Paradigms  
- Objects and Classes  
  - Data Members, Access Specifiers  
  - Arrays & Array of Objects  
  - Constructors (Default & Parameterized)  
  - Static and Constant Members  
- Object-Oriented Design with UML  
  - Class, Object, and Sequence Diagrams  
- Java Environment Setup  

## **Unit 2: Advanced Java and Object-Oriented Concepts**
- Polymorphism and Inheritance  
- Interfaces and Abstract Classes  
- Packages and Exception Handling  
- Java Libraries and Utilities  
  - Numbers, Math, Strings, StringBuilder, Regex  
  - File Handling: Accessing and Modifying Files  
  - Generics and Collections  

## **Unit 3: Concurrency, GUI, Networking, and Principles**
- Introduction to Threads  
  - Creating Threads, Thread States, Synchronization  
- Serialization in Java  
- Basics of GUI Programming  
- Java Sockets and JDBC  
- SOLID Principles in Object-Oriented Design  
- UML for Software Modeling  

## **Textbooks**
- **Primary Textbook:** *Java How to Program* by Paul Deitel & Harvey Deitel  
- (*School-suggested textbooks are less recommended for practical exercises*)
